# What is this?
A set of collated and new configs for various mods, mainly Tantares (+LV, SP and SAF), in the same vein as NFRealism. Some configs will break pre-existing craft files. 
Pretty much all of these haven't been tested with or made with standard gameplay progression in mind, maybe in future. Be mindful for now.

Designed to be downloaded as a whole then unwanted patches removed, since some require the "Kea_configs" file path.

Currently incompatible with Chemical Technologies.
