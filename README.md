# What is this?
A set of collated and new configs and partswitches for various mods, mainly Tantares (+LV, SP & SAF), in the same vein as NFRealism. Some configs will break pre-existing craft files, particularly the rescale patches. Stats have also been tweaked for many Tantares engines so be aware those upper stages might be a bit underpowered now.
Pretty much all of these haven't been tested with or made with standard gameplay progression in mind, maybe in future.

Designed to be downloaded as a whole then unwanted patches removed, since some require the "Kea_configs" file path.

Currently incompatible with Chemical Technologies.
